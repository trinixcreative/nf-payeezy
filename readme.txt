== Description ==
The Ninja Forms Payeezy Extension allows for the inclusion of credit card processing via the Payeezy Gateway.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `ninja-forms-payeezy` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the 'Forms' menu item in your admin sidebar
4. When you create a form, you can now add payment fields on the field edit page.
5. A "Payeezy Settings" link will now appear underneath the main "Forms" admin menu.
